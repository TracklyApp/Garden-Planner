# Garden Planner & Tracker

A single-file garden planner and tracker that runs entirely in your browser — no server, no account, no build step. Just open `garden-planner.html`.

## Features

- **Dashboard** with season goals, live stats, and a "This Week" digest of overdue/upcoming tasks and planting dates
- **Garden Layout**: sketch upload, bed list with photos and dimensions, and automatic companion-planting conflict warnings between neighboring beds
- **Garden Log**: Seed Inventory, Planting Schedule (with frost-date warnings), Watering Log, Fertilizer Log, Pest & Disease Tracker, Weather Notes, Plant Growth Tracker, Harvest Tracker, Garden Expenses, Produce Valorization, Yield Summary (with per-crop cost/profit breakdown and a monthly harvest chart), and Garden Journal
- **Bonus Pages**: Companion Planting Chart, Frost Dates Tracker, Monthly Garden Planner, Seed Starting Tracker, Harvest Calendar, Photo Gallery, and Garden To-Do List (auto-synced with Planting Schedule)
- **Global search** across every section
- **Dark mode**, print-friendly stylesheet, and full mobile responsiveness
- **Export/Import** full JSON backup
- **Preview mode**: a limited number of free changes before an activation code is required, for distributing as a demo

## Getting started

1. Download `garden-planner.html`.
2. Open it in any modern browser (double-click, or drag it into a browser window).
3. Your data is saved locally in that browser via `localStorage` — nothing leaves your machine.

## How to use

The full user manual lives in [`garden-planner-manual.html`](garden-planner-manual.html) — also linked from inside the app itself (❓ Manual button).

## Data & privacy

All data stays in your browser's local storage. Clearing your browser's site data for this local file will erase it — use **Export** regularly to keep a backup.

## Tech

Plain HTML/CSS/JavaScript, zero dependencies, zero build step.
